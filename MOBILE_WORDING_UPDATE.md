## 🚀 文案策略总结

### 英文文案策略

1. **用简单的词** - 避免复杂词汇
   - "Catch" 代替 "Detect"
   - "Pick" 代替 "Choose"
   - "Find" 代替 "Identify"

2. **去掉多余的词** - 删除冠词、形容词
   - "Upload image" 代替 "Upload an image"
   - "5 Free Daily" 代替 "5 free checks daily"

3. **短句** - 每句不超过5-7个词
   - "Images, Voices, Videos - All Covered"
   - "Start Free, Upgrade Anytime"

4. **有力的动词** - 用行动词
   - "Catch", "Spot", "Find", "Stop"

---

### 中文文案策略

1. **去掉冗余** - 删除不必要的字
   - "验真假" 代替 "验证内容真实性"
   - "抓假号" 代替 "识别虚假用户"

2. **口语化** - 用日常用语
   - "试试" 代替 "试一试"
   - "搞定" 代替 "完成"

3. **短句** - 每句2-5个字
   - "1秒识破"
   - "快速发现"

4. **有冲击力** - 用强烈的词
   - "识破" 代替 "识别"
   - "抓" 代替 "发现"
